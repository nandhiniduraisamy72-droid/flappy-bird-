# 🐤 Flappy Bird - Single File Game

## 📌 Project Description
This is a simple **Flappy Bird game** built using only **HTML, CSS, and JavaScript** in a single file.  
No external frameworks or backend are used.

The game runs directly in the browser.

---

## 🚀 Features
- Classic Flappy Bird gameplay
- Smooth animations (bird, pipes, clouds, ground)
- Score tracking system
- Best score saved using Local Storage
- Game Over screen with medals 🏆
- Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used
- HTML5  
- CSS3 (Animations, Gradients, Flexbox)  
- JavaScript (Vanilla JS)  

---

## 🎮 How to Play
- Click **START** button  
- Press **Spacebar** or **Mouse Click** to jump  
- Avoid hitting pipes and ground  
- Try to get the highest score  

---
Contains:
- HTML (structure)
- CSS (design & animations)
- JavaScript (game logic)

---

## 💾 Storage
- Best score is saved using **LocalStorage**

---

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. Start playing 🎉
